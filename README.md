# دليل المنتجات البرمجية الرقمية الجاهزة (Premium Coding Solutions)

مجموعة متكاملة من السكريبتات والحلول البرمجية الجاهزة للبيع والاستخدام المباشر في مجالات هندسة البيانات، الأمن السيبراني، أتمتة الأنظمة، وهندسة الشبكات.

---

## 📦 المنتج 1 و 2: نظام أتمتة وهندسة البيانات المتكامل (ETL Pipeline & Docker)
- **الوصف:** نظام لسحب وتطهير ورفع البيانات تلقائياً من ملفات CSV وإزالة التكرار والأخطاء عبر Pandas ورفعها لقاعدة بيانات PostgreSQL داخل حاويات Docker المستقلة.
- **الملفات:** `app.py`, `docker-compose.yml`, `customers.csv`, `products.csv`, `orders.csv`

---

## 📦 المنتج 3: نظام الفحص الأمني المؤتمت لثغرات XSS (Cybersecurity Scanner)
- **الوصف:** أداة برمجية متقدمة تفحص المواقع تلقائياً وتكشف ثغرات حقن البيانات (XSS Reflected) عبر النماذج وحقول الإدخال، وتدعم تقنية تعدد الخيوط (Multi-threading) لتوليد تقارير أمنية فورية.
- **الملفات:** `security_scanner.py`

---

## 📦 المنتج 4: سكريبت أتمتة ومراقبة خوادم لينكس (Linux Server Monitor & Backup)
- **الوصف:** سكريبت مراقبة حية لأداء السيرفر واستهلاك الموارد (CPU, RAM, Disk) عبر مكتبة `psutil` مع أتمتة كاملة لأخذ نسخ احتياطية مضغوطة لسجلات النظام بصيغة (`tar.gz`) لحمايتها وتأمينها.
- **الملفات:** `server_automation.py`

---

## 📦 المنتج 5: حزمة تصفية ومعالجة إشارات الشبكات الرقمية (MATLAB Signal Filtering)
- **الوصف:** مشروع هندسي تطبيقي على MATLAB لمحاكاة استقبال الإشارات اللاسلكية وتطبيق فلاتر رقمية متطورة (Low-Pass Butterworth Filter) لعزل التشويش والضوضاء لضمان استقرار نقل البيانات.
- **الملفات:** `network_filter.m`
